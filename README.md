Nama App: Pendaftaran Stock Bahan Pokok
Tujuannya: Aplikasi yang dapat meregistrasi, menghapus, mengedit banyak jenis bahan pokok
Cara menggunakan: Jalankan node index.js di command line, kemudian
buka localhost:3000/register - untuk menambah barang
buka localhost:3000/display - untuk melihat list barang, menghapus, dan mengubah data barang
